## Database Schema

### Core Tables
- **members** - User profiles with geographic data
- **records** - System/module records
- **community_power** - Contribution tracking
- **businesses** - Business registry
- **creators** - Creator profiles
- **experiences** - Events and experiences
- **delivery_bookings** - Delivery orders
- **riders_drivers** - Logistics personnel
- **sika_records** - Trust and value records
- **verification_badges** - Verification levels
- **readiness_requests** - Regulatory/provider requests
- **audit** - Action logging

### Infrastructure Tables
- **dispatch_jobs** - Delivery dispatch
- **rider_status** - Rider/driver status
- **mail_items** - Internal messaging
- **notifications** - System alerts
- **navigation_routes** - Route tracking
- **map_places** - Geographic markers
- **weather_records** - Weather data
